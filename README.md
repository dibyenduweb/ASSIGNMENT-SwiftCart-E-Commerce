## 1. Difference Between null and undefined

**undefined** Automatically assigned to variables that are declared but not initialized.

**null** Must be assigned to a variable.

## 2. map() vs forEach()

**map()** when you need to transform the data in an array and want a new array containing the transformed values, leaving the original array unchanged.

**forEach()** j when you need to iterate over an array to perform a side effect or an action for each element, and you don't need a return value or a new array.

## 3. Difference Between == and ===

**==** == checks only value .type can be different.

**===** === checks value and type must be same.

## 4. Why async/await is Important for API Calls

**async/await** async/await makes API code clean, readable, and easier to handle errors.

## 5. Understanding Scope in JavaScript

Scope is the area where a variable is accessible in a program.
JavaScript has 3 scopes — Global, Function, and Block.

### Global Scope

Variables declared outside any function — usable everywhere.

### Function Scope

Variables declared inside a function — usable only inside that function.

### Block Scope

Variables declared with `let` or `const` inside `{ }` — usable only inside that block.
